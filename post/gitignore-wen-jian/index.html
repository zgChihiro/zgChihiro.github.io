.gitignore文件

Posted on 2017-10-18，1 min read

.gitignore就是忽略不必要的提交文件

如图所示的这个zip文件就不会加入版本管理

下面介绍如何创建 ".gitignore" 文件以及如何使用

首相进入git版本库

在根目录下鼠标右键 Git Bash Here

然后输入 touch .gitignore 在文件夹就生成了一个“.gitignore”文件。

输入“*.zip”git库中的所有zip文件都会被过滤
输入“/hello/”git库会过滤掉 hello 这个文件夹
输入 “/hello/a.txt” git会过滤掉指定的文件

我们创建出一堆的zip文件

不会加入git的管理

**

需要注意的是. gitignore必须在git添加版本之前，也就是如果git已经对文件进行管理了的话，后期即使添加到gitignore也是没有用的
